# 프로젝트 소개
'Private Stay'는 개인 숙박시설 예약을 위한 프로그램입니다. 이 프로그램은 예약 과정을 자동화하여 간소화하고, 
전용 숙박 예약을 통해 프라이버시와 프리미엄 경험을 제공하며, 지역 숙박 사업과 협력을 통해 COVID-19 이후 지역 경제를 활성화하는 것을 목표로 합니다
